## Key Insights from Two-Round External Review

### 1. External Review is Non-Negotiable

**Lesson**: Internal review cannot replace external fact-checking. The 2.9-point gap (9.4 → 6.5) was structural, not personnel.

**System fix**: Added Phase 3.5 (External Verification) to Content Factory v2.1.

### 2. Strawman Opponents Weaken Arguments

**Lesson**: Attacking weak positions makes your argument look weaker. Steelmanning creates trust.

**System fix**: Editor agent now checks for strawman opponents as critical issue.

### 3. Precision Errors Undermine Credibility

**Lesson**: One wrong date/number can sink entire piece. Readers stop trusting when they catch errors.

**System fix**: External verification checks all named entities, numbers, and absolute claims.

### 4. Terminology Clarity Matters

**Lesson**: Technical terms ("ontology") confuse readers if undefined or misused.

**System fix**: Either use clear terminology OR define technical terms explicitly.

### 5. "Composite" Examples Need In-Line Labels

**Lesson**: Footnote disclaimers aren't enough. Readers will treat examples as real without prominent labeling.

**System fix**: All composite/illustrative examples labeled at first mention with **bold** text.
