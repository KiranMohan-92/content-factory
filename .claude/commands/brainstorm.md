# Brainstorm: Multi-Agent Topic Research

Generate article ideas for content factory v.2 using a two-stage AI pipeline:
1. **Gemini 2.5 Flash** - Research trending topics
2. **OpenCode GLM 4.7** - First principles analysis with hard-to-vary explanations

## Configuration

- **Output folder**: `/mnt/c/Users/kiran/myprojects/content-factory/brainstorm-outputs/`
- **Gemini model**: `gemini-2.5-flash` (with `-y` for headless, `-o text` for output)
- **OpenCode model**: `opencode/glm-4.7-free`

## Task

### 1. Parse Arguments

If "$ARGUMENTS" is provided, use it as topic areas.
Otherwise, default to: "productivity, personal development, AI adaptation"

### 2. Generate Timestamp

Create filename with today's date: `brainstorm-YYYY-MM-DD.md`

### 3. Stage 1: Gemini Research

Run Gemini CLI in headless mode:

```bash
gemini -m gemini-2.5-flash -y -o text "Research and identify the current trending topics in these areas:

$TOPIC_AREAS

For each area, provide:
- Top 5-7 trending subtopics with brief explanations
- Why they're trending now (cultural/technological drivers)
- Potential article angles that would resonate with readers

Focus on actionable, practical topics that would make compelling content for thoughtful readers interested in self-improvement and technology."
```

**If Gemini quota is exhausted**: Wait for quota reset or notify user.

Save raw research to: `brainstorm-outputs/gemini-research-YYYY-MM-DD.md`

### 4. Stage 2: OpenCode First Principles Analysis

Run OpenCode with GLM 4.7:

```bash
opencode run -m opencode/glm-4.7-free "Analyze this trending topics research using FIRST PRINCIPLES THINKING and HARD-TO-VARY EXPLANATIONS.

[INSERT GEMINI RESEARCH OUTPUT]

YOUR TASK - Apply First Principles and Hard-to-Vary Explanations:

1. FIRST PRINCIPLES DECONSTRUCTION:
   - Break down each trend to its fundamental, irreducible components
   - What are the root causes that MUST be true for these trends to exist?
   - Strip away buzzwords - what's the actual underlying human need or technological capability?

2. HARD-TO-VARY EXPLANATIONS:
   - Which explanations for these trends are robust and would be hard to modify without becoming wrong?
   - Identify explanations that are TOO easy to vary (vague, unfalsifiable, or could explain anything)
   - What would DISPROVE each trend being genuinely important?

3. CRITICAL ANALYSIS:
   - What's missing? What's overhyped?
   - What are the deeper, non-obvious trends underneath these surface topics?
   - Which trends are symptoms vs root causes?

4. CONTRARIAN ANGLES:
   - Identify 5-7 contrarian article ideas that challenge conventional wisdom
   - Use first principles to justify why the contrarian view might be correct

5. INTERSECTION OPPORTUNITIES:
   - Find unique crossover topics between the topic areas
   - Apply first principles to identify NON-OBVIOUS connections

6. TIMELESS vs TRENDY:
   - Use hard-to-vary explanations to distinguish which topics have fundamental staying power
   - Which are hype cycles that fail first principles tests?

7. CONTENT FACTORY RECOMMENDATIONS:
   Based on first principles analysis, recommend 10 specific article ideas optimized for:
   - Depth over clickbait
   - Hard-to-vary explanations over vague claims
   - Practical implementation over theory
   - Long-term value over trending keywords
   - Original thinking that survives first principles scrutiny

Format as structured markdown."
```

## Error Handling

- **Gemini quota exhausted**: Report time until reset, suggest waiting or using alternative model
- **OpenCode unavailable**: Fall back to direct analysis without GLM
- **Network issues**: Retry up to 3 times with exponential backoff

## Notes

- This skill is designed for brainstorming and ideation, not content production
- Use generated ideas as inputs to the main content factory pipeline
- First principles analysis helps filter trendy hype from timeless insights
